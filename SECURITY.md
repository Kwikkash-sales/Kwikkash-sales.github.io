# Security Policy

## 🔐 Supported Versions

Only the latest version of Kwik Kash Store is supported.

## 🚨 Reporting a Vulnerability

If you discover a security vulnerability in this project, please DO NOT open a public issue.

Instead, report it privately via email:

📧 sales@kwikkash.store

Please include:
- A description of the issue
- Steps to reproduce it
- Any potential impact

We will review and respond as quickly as possible.

## ⚠️ Policy

Unauthorized testing, scanning, or exploitation of this site or its infrastructure is strictly prohibited.
